# Velofolio CRM: The Grand Master Manual (Strategic Edition)

This manual is the ultimate functional audit and strategic reference for the Velofolio CRM ecosystem. It provides a comprehensive catalog of every page, interactive element, and specific professional refinement built into the platform.

## 1. Professional Platform Audit & Strategic Roadmap
The Velofolio CRM has been meticulously engineered to meet the highest standards of studio management technology, incorporating extensive architectural refinements and a future-focused integration plan.

### 1.1 UI & Layout Architecture
- **Dashboard Optimization**: The central graph has been adjusted for improved visual hierarchy, the layout has been refined, and the overall dashboard width has been reduced to maximize data density.
- **Navigation Design**: The studio logo has been increased in size, the navbar height has been expanded, and the active navigation tab is now professionally highlighted.
- **Typographic Precision**: Global font sizes have been lowered for a more high-end appearance, and interactive labels now utilize bold styling instead of deletion icons for a cleaner interface.
- **Spatial Padding**: Additional whitespace has been added to the base of every page to ensure comfortable scrolling on large-format displays.

### 1.2 Core Module Functionality
- **Dashboard Management**: Performance data visualization focuses exclusively on Leads, Shoots, and Revenue for absolute operational focus.
- **Standardized Color Coding**: Standardized professional color coding—**Red for Leads** and **Green for Jobs**—is implemented across all modules.
- **Client & Company Registry**: Full implementation of the Timezone selector, specialized "Person Name" fields for corporate clients, and Date Created sorting. Tabular views have been streamlined by removing redundant columns.
- **Production Controls**: Card names are fully customizable, 20MB file attachment thresholds are established, and "Mark-to-Update" status buttons are active for rapid workflows.
- **Leads Pipeline**: The Lead detail screen utilizes the active Job workflow architecture. Booked leads are automatically archived from the active sales view to maintain focus on new inquiries.
- **Calendar Scheduling**: Scheduling now lists events based on the actual **Event Date** rather than the initial booking date, ensuring precise operational coordination.
- **Financial Oversight**: Payment filters focus exclusively on status, with dashboard-style date range selection now integrated into the payments ledger.

### 1.3 Strategic Integration Phases
The current phase of platform development focuses on deepening the backend logic and integrating advanced communication systems:
- **Relational Data Integrity**: Engineering absolute technical links ensure Clients are dynamically tethered to their respective Jobs, Invoices, and Leads across the entire ecosystem.
- **Studio-Integrated Messaging**: Implementation of a professional, integrated mailing engine for unified, template-based client communication.
- **SuperAdmin Audit Tracks**: Development of a sophisticated notification layer to notify SuperAdmins of any modifications performed by the team for full studio accountability.
- **Unified Template Hub**: Centralizing all Products, Packages, Contracts, Quotes, and Invoices into a global template management hub within Settings.
- **Automated Financial Aggregation**: Engineering real-time total calculations for Paid and Unpaid amounts that update dynamically according to active filters.
